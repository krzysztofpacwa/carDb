# carDb

## General
API with UI in Thymeleaf. Car as a model of application.

Features:
* CRUD
* Filter by color
* Filter from/to date

## Technologies

- Java 11

- Maven

- Spring Boot 2.4.0

- Thymeleaf

- Bootstrap(CSS)

- JQuery/JavaScript

- MySQL (remotesql.com)

 **Design patterns**
- Model-view-controller (MVC) 
- Data Access Object (DAO)
